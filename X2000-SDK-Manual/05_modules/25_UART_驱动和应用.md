# 1. 模块简介

通用异步接收器/发送器（UART）串行端口。 共有十个UART：所有UART使用相同的编程模型。 每个串行端口都可以在基于中断的模式或基于DMA的模式下运行。通用异步接收器/发送器（UART）与16550行业标准兼容，可以用作符合红外数据协会（IrDA）串行红外规范1.1的慢速红外异步接口。
